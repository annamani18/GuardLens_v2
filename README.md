# GuardLens v2 🛡️

AI Chrome Extension that detects dark patterns in e-commerce websites and gives trust scores.

## Features
- Trust Score System
- Fake Urgency Detection
- False Scarcity Detection
- Market Comparison
- Report Issue System
- Real-time Re-scan

## Tech Stack
JavaScript, HTML, CSS, Chrome Extensions API

## Installation
1. Download repo
2. Open chrome://extensions
3. Enable Developer Mode
4. Load Unpacked
5. Select folder
